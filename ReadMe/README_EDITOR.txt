# README_EDITOR（編集者向け：Git不要で更新する）

このREADMEは「GitやHTMLが分からなくても、GUIで劇団サイトを更新できる」ための手順です。  
あなたがやるのは **editor.html を開いて編集 → ZIPを書き出して管理者に渡す** だけです。

---

## 1. 管理者から受け取るもの

最低限これが入っているフォルダを受け取ってください：

- `editor.html`
- `editor.js`
- `editor.css`

あると便利（ほぼ必須レベル）：
- `content/site.json`（今のサイト内容）
- `assets/uploads/manifest.json`（画像一覧：写真を選ぶため）
- `assets/uploads/`（既存画像：確認用。無くても更新はできます）

---

## 2. あなたがやる作業（超ざっくり）

1) `editor.html` を開く（Chrome推奨）  
2) （任意）「JSONを読み込む」で `content/site.json` を読み込む  
3) （推奨）「画像一覧を読み込む」で `assets/uploads/manifest.json` を読み込む  
4) 文章や日程をフォームで更新  
5) 写真を差し替える（必要な場合）  
6) 「ZIPを書き出す（画像込み）」を押す  
7) 出てきた `update_pack_YYYYMMDD.zip` を管理者に送る

---

## 3. 画面のボタンの意味

- **JSONを読み込む**：前回の `site.json` を読み込んで “続きから編集” できます
- **画像一覧を読み込む（manifest）**：写真選択の候補を出すための一覧です（おすすめ）
- **テンプレに戻す**：初期状態に戻します（間違って押したら、もう一度 JSON を読み込めばOK）
- **ZIPを書き出す（画像込み）**：これがメイン。管理者に渡すファイルを作ります  
- **JSONを書き出す**：文章だけ変えたい時用（画像更新が無い時）。基本はZIPでOK

---

## 4. メンバー写真をGUIで差し替える方法

### 4-1) 既存画像から選ぶ（いちばん簡単）
1) 先に **「画像一覧（manifest.json）」を読み込む**  
2) 「メンバー」欄で、対象メンバーの **写真（Photo）** をドロップダウンから選ぶ  
3) 右側プレビューで JSON が更新されているのを確認

### 4-2) 新しい画像を追加して選ぶ（差し替え）
1) 画面内の **「画像を追加（uploads）」**（または同等のボタン）で画像を選ぶ  
2) 追加した画像が **写真の候補**に出てくるので選ぶ  
3) 最後に **ZIPを書き出す**  
→ ZIPの中に `assets/uploads/あなたの画像` が入ります（管理者が取り込みます）

> ✅ ファイル名は **半角英数・アンダースコア推奨**（例：`member_taro.jpg`）  
> ✅ 画像サイズは 1枚あたり **1MB以下**くらいが目安（重いと表示が遅くなります）

---

## 5. 送る前のチェック（おすすめ）
- 誤字脱字
- Newsの日付が正しい（例：2025-12-28）
- 予約リンクなどのURLが正しい（https:// で始まる）
- 写真が“意図した人”になっているか

---

## 6. 管理者へ送るもの
- **`update_pack_YYYYMMDD.zip`（これ1つだけでOK）**

送付手段：Slack / Google Drive / メール なんでもOK。

---

## 7. よくある質問

### Q) editor.html を開いたのに画像一覧が出ない
A) **manifest.json を読み込んでいない**可能性が高いです。  
「画像一覧を読み込む」で `assets/uploads/manifest.json` を選んでください。

### Q) 写真を追加したのに候補に出ない
A) 追加直後に候補に出ない場合は、いったん別の値を選んでから戻すか、再読み込みしてみてください。  
それでもダメなら、管理者にZIPを送って「この画像を使いたい」と伝えればOKです。
